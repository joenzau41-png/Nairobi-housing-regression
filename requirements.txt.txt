## How to Run

**1. Clone the repository**
git clone https://github.com/joenzau41-png/nairobi-housing-regression.git
cd nairobi-housing-regression

**2. Install dependencies**
pip install -r requirements.txt

**3. Run the notebook**
jupyter notebook notebooks/housing_regression.ipynb

## Tools Used

- **Python 3.11**
- **pandas** — data cleaning and feature engineering
- **statsmodels** — OLS regression with full statistical summary
- **scikit-learn** — train/test split and model evaluation
- **seaborn / matplotlib** — EDA and result visualisation
- **joblib** — model persistence
